## 启动
```
./nginx
```

## 重启
```
./nginx -s reload
```

## 停止
##### 方法1：
查看nginx主进程号
```
ps -ef | grep nginx
```
master进程则为主进程，它的编号就是主进程号，如图：

![查看nginx进程](https://img.yangrunwei.com/article-img/20160226/feeff68c-0ff2-4935-b8dc-38906e39f12b--33-1.png "查看nginx进程")

然后执行下面命令停止nginx
```
kill -QUIT 1215
```
