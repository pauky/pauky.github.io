##jade混合文字与标签
当我们在使用jade时，如果标签中包含的文字过多，为了美观，我们会这样做
```
p
	| 第一段好多好多文字···
	| 第二段好多好多文字···
	| 第三段好多好多文字···
```
这时如果我们想给上述文字中的“第二段”加链接，可以这样做：
#### 方法1
```
p
	| 第一段好多好多文字···
	a(href="#") 第二段
	| 好多好多文字···
	| 第三段好多好多文字···
```

####方法2
```
p.
	第一段好多好多文字···
	<a href="#">第二段</a>
	好多好多文字···
	第三段好多好多文字···
```
注：p标签后的“.”不可省略。
